About the app

Project information

  • Project Name: Ethiopian Sign Language Interpreter
  • Category: Mobile App
  • BahirDar University: Final year preject
  • Project Date: july, 2023

The Ethiopian Sign Language Interpreter is a cutting-edge technology that facilitates seamless communication by recognizing and translating EthSL signs into amharic text. this project has been recognized with the Best Project Award from Bahir Dar University, acknowledging its innovation and impact in facilitating communication for the hearing-impaired community.
